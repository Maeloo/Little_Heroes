Detailed Description

replacement for the Unity Rect class that is TouchKit and resolution-aware.

Creating one will automatically scale all values if TouchKit autoScaleRectsAndDistances is true based on your designTimeResolution.

Note the TKRects use the bottom-left as the origin.
