void GUICustomAuth.OnCustomAuthenticationFailed ( string  debugMessage)

This method is called when Custom Authentication is setup for your app but fails for any reasons.

Unless you setup a custom authentication service for your app (in the Dashboard), this won't be called. If authentication is successful, this method is not called but OnJoinedLobby, OnConnectedToMaster and the others will be called.
