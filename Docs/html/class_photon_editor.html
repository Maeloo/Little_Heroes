bool PhotonEditor.dontCheckPunSetup
staticgetsetprotected

Can be used to (temporarily) disable the checks for PUN Setup and scene PhotonViews.

This will prevent scene PhotonViews from being updated, so be careful. When you re-set this value, checks are used again and scene PhotonViews get IDs as needed.
